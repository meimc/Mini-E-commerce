🛒 PROYECTO: Mini E-commerce (Carrito de Compras)

📌 Enunciado

Crear una tienda virtual donde el usuario pueda visualizar productos, agregarlos al carrito y simular una compra.

🎯 Objetivo

Manejo de eventos, arrays de objetos, renderizado dinámico y lógica de negocio.

⚙️ Requerimientos

Funcionales:
* Mostrar catálogo de productos (JSON simulado)
* Agregar/quitar productos al carrito

Mostrar:
* Total de compra
* Cantidad de productos
* Modificar cantidades
* Simular checkout (mensaje de compra exitosa)

No funcionales:
* Uso de Bootstrap (cards, navbar, badges)
* Interfaz clara y moderna
* Código reutilizable

---
*Extras:*
* Guardar carrito en localStorage
* Filtro por categoría o precio
* Simulación de API con fetch (JSON local)
